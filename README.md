Python3, Pandas, Argparse, Numpy, Scipy, Progressbar

- Windows : It also works on Windows.
### Python Packages :
- Pandas
- Argparse
- Numpy
- Matplotlib
- Scipy
- Progressbar
### Data files :

`git clone https://github.com/hocinebib/Projet_Court_H-H.git`

2. Runing the code :
`python3 main.py pdbfile nbr_of_pts`
